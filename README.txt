GitHub Pages 更新方式：
1. 將本資料夾內所有檔案上傳到原 repository 根目錄。
2. 確認 index.html、styles.css、app.js、sw.js 位於最外層。
3. Commit changes 後等待 GitHub Pages 重新部署。
4. 手機若仍顯示舊版，先用 Safari 開啟網址重新整理；必要時刪除主畫面舊捷徑再加入。
